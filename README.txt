Project Name: Impressionist
Author: Al-Latif Alston
Description: A simple, custom-made doodling application.

- DOWNLOAD IMAGES: downloads images from an online directory
- LOAD IMAGE: loads an image from gallery
- SAVE IMAGE: saves image to the gallery
- BRUSH: choose brush type
    - CIRCLE: circular, velocity-based
    - SQUARE: square, velocity-based
    - ILLUMINATI: triangular border, dot in middle
    - SPRAY PAINT: spray paint brush effect
- INVERT: swaps RGB values
- CLEAR: clears drawing area

KNOWN BUGS:
    - When drawing, crashes if cursor goes outside canvas.
